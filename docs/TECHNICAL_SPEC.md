# Техническое задание RICH HALL Minsk

## Формат и цель

Адаптивный одностраничный landing page для презентации банкетного пространства, демонстрации залов и получения заявок на бронирование.

## Обязательные секции

- главный экран;
- форматы мероприятий;
- залы;
- галерея;
- преимущества;
- отзывы;
- форма бронирования;
- контакты.

## Функции

- адаптивная верстка;
- навигация по секциям;
- отправка заявки;
- карта;
- кнопки связи и социальных сетей;
- интерфейсные анимации.

## Поля формы

- имя;
- номер телефона;
- тип мероприятия;
- количество гостей;
- дата мероприятия;
- комментарий.

После успешной отправки должно отображаться подтверждение.

## Технологии и дизайн

Сайт реализуется на HTML, CSS и JavaScript, корректно работает на мобильных устройствах, планшетах и desktop-экранах. Визуальный стиль — минималистичный premium, соответствующий тематике банкетного пространства.

## Панель администратора

Административная панель доступна по отдельной ссылке `/admin/` и предусматривает вход по логину и паролю. В админке реализовано управление залами:

- создание новых залов;
- изменение информации о существующих залах;
- удаление залов;
- редактирование названий и описаний;
- добавление, замена и удаление фотографий залов;
- сохранение и публикация изменений на сайте.

После сохранения обновлённая информация автоматически подтягивается на основной сайт.

## Результат

Заказчику передаются готовый опубликованный сайт и исходный код проекта.
